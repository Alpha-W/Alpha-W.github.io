Leetcode 7:Reverse Integer

By Hau Wong, 六 12 五月 2018, in category 算法

Description

Get.

class Solution:
 def reverse(self, x):
     """
     :type x: int
     :rtype: int
     """
     if x<0:
         t=-x
         return -(self.reverse(t))
     else:
         s=int((str(x))[::-1])
         if s<=0 or s>=2**31:
             return 0
         else:
             return s
