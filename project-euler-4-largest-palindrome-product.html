Project Euler 4: Largest palindrome product

public class Main {
public static boolean isPalindrome(int p)
{
 int digit=1+(int)Math.log10(p);
 int[] a=new int[digit];
 for(int k=0;k<digit;k++) a[k]=((p/(int)Math.pow(10,k))%10);
 for(int k = 0; k < digit; k++)
     if(a[k]!=a[digit-1-k]) return false;
     return true;
}
 public static void main(String[] args) {
     int i,j,s=0;
     for (i = 100; i <1000 ; i++)
         for (j = 100; j <1000 ; j++)
             if(isPalindrome(i*j)) s=(s>i*j?s:(i*j));
             System.out.println(s);
 }
}//906609

Comments

欲计算整数 \(p\)在 \(s\)进制下的各项系数 \(a_{k}\)，即求 \(p=\sum_{k=0}^{n}a_{k}s^{k}\)中的 \(a_{k},k=0,...,n\),可以有以下两种方法：

  1. \(a_{k}=p/s^{k}%s\)
  2. \(a_{k}=p%s^{k+1}%s^{k}\)

注：这里"/"为整数除法，"%"为整数求余，即 \(a/b:=-sgn(\frac{a}{b})\lceil -\mid \frac{a}{b}\mid \rceil\), \(a%b:=a-b*(a/b)\)

这里选用第一种，第二种不好的地方在于即使 \(p\)不溢出， \(s^{k+1}\)仍然可能会溢出。
